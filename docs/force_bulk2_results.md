## force bulk2 results

Get job results

```
force bulk2 results <jobId> [flags]
```

### Options

```
  -f, --failed        Show failed results only
  -h, --help          help for results
  -s, --successful    Show successful results only
  -u, --unprocessed   Show unprocessed records only
```
