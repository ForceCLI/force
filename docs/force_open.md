## force open

Open a browser window, logged into an authenticated Salesforce org

### Synopsis


Open a browser window, logged into an authenticated Salesforce org.
By default, the active account is used.


```
force open [account] [flags]
```

### Examples

```

  force open user@example.com

```

