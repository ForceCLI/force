## force bulk2 hardDelete

Hard delete records using Bulk API 2.0

```
force bulk2 hardDelete <object> <file> [flags]
```

### Options

```
  -i, --interactive         Interactive mode (implies --wait)
      --lineending string   Line ending (LF or CRLF) (default "LF")

```
